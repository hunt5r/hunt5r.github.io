title:hive使用笔记

###1.This command is not allowed on an ACID table

hive在使用select，insert,update等查询查询时抛出异常 This command is not allowed on an ACID table

解决办法（执行如下命令）：
hive> set hive.txn.manager=org.apache.hadoop.hive.ql.lockmgr.DbTxnManager;
hive> set hive.support.concurrency=true;
hive> set hive.enforce.bucketing=true;
hive> set hive.enforce.bucketing=nonstrict;
hive> set hive.compactor.initiator.on=true;
hive> set hive.compactor.worker.threads=2;

###2.flume同步至数据至hive表中部分字段为null

解决办法：检测字段是否存在大小写问题，如果存在全部修正为小写

###3.flume同步数据至hive注意事项

Hive表必须设置bucket并且 stored as orc

example:
    CREATE TABLE table_name (
        id                int,
        name              string
    )
    CLUSTERED BY (id) INTO 2 BUCKETS STORED AS ORC
    TBLPROPERTIES ("transactional"="true",
        "compactor.mapreduce.map.memory.mb"="2048",
        "compactorthreshold.hive.compactor.delta.num.threshold"="4",
    "compactorthreshold.hive.compactor.delta.pct.threshold"="0.5"
    );
